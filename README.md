# SupportLens AI

SupportLens AI is an AI-powered Enterprise IT Incident Analyzer that assists IT support engineers in diagnosing technical issues using Google's Gemini AI.

---

## Features

- AI-powered incident analysis
- Screenshot analysis
- System log analysis
- Root cause identification
- Recommended troubleshooting steps
- Severity estimation
- Confidence scoring

---

## Tech Stack

### Backend
- Python
- FastAPI
- Google Gemini API

### Frontend
- HTML
- CSS
- JavaScript

## How to Run

Clone the repository

```bash
git clone https://github.com/Suggestmeanamethen/SupportLens-AI.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run

```bash
uvicorn backend.app:app --reload
```

Open

```
http://127.0.0.1:8000
```
